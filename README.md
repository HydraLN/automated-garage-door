<h3>
  Getting Started
  </h3>
  <li>Download kode</li>
  <li>Library</li>
  <li>Mencari port dan board</li>
  <li>Diagram circuit</li>


  <h3>Download kode</h3>
  <p>Langkah pertama adalah mendownload kode</p>
  <span>
  1. Klik sketch_jan31a.ino <br>
  2. Klik Download rawcode/ copy rawcode <br>
  3. Masuk ke arduino IDE dan paste kode/buka sketch dengan file yang telah di downlaod
  </span>

  <h3>
    Library
  </h3>
  <p>Mendownload library untuk servo motor</p>
  <span>
    1. Navigasi ke bagian kiri layar dan klik Library Manager <br>
    2. Cari Servo by Michael <br>
    3. Pilih Install All
  </span>

  <h3>
    Mencari port dan board
  </h3>
  <p>Pastikan Driver sudah terintall untuk port, yang biasanya secara otomatis terunduh dengan Arduino IDE <br>
    </p>
    
 > Untuk board clone biasanya menggunakan driver *CH340* dan harus mengunduh driver secara manual<br>

<span>
   Cara unduh Driver secara manual <br>
   https://docs.arduino.cc/tutorials/generic/DriverInstallation/ <br>
</span>

<span>
  1. Hubungkan board ke komputer menggunakan USB Cable <br>
  2. Pilih PORT contoh: (COM 3, COM8) <br>
  3. Pilih Board Arduino UNO <br>
  4. Upload Sketch <br>
</span>
